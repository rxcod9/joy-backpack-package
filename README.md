## Installation

```bash
# 1. Require this Package in your fresh Laravel/Backpack project
composer require joy/replace-keyword

# 2. Publish
php artisan vendor:publish --provider="Joy\ReplaceKeyword\ReplaceKeywordServiceProvider" --force
```
